# Laboratorio 05 Proyecto: Modelado de Procesos de Sodimac

## Descripción
Este proyecto tiene como objetivo modelar, analizar y documentar los procesos de negocio clave de la empresa **Sodimac** utilizando la notación **BPMN (Business Process Model and Notation)**.

Se representan flujos de trabajo, áreas involucradas, entradas, procesos y salidas, con el fin de comprender la operación de la empresa y proponer mejoras en la gestión de procesos.

## Objetivos del proyecto
- Modelar procesos empresariales usando BPMN
- Analizar la interacción entre áreas y roles
- Identificar oportunidades de mejora en los procesos
- Aplicar conceptos de Ingeniería de Software Empresarial

## Empresa seleccionada

**Nombre:** Sodimac  
**Sitio web:** https://www.sodimac.com.pe  

**Descripción:**  
Empresa dedicada a brindar soluciones integrales para el hogar, construcción y mejoramiento de espacios.

## Procesos modelados
- Gestión de la Cadena de Suministro y Logística  
- Gestión de Servicios al Cliente  
- Gestión de Postventa y Experiencia del Cliente  
- Gestión de Responsabilidad Social y Sostenibilidad
